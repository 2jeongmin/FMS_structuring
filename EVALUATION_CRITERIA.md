# 전담시설물 평가 기준

## A. 안전관리 필요성 (70점)

### 1. 기능성 (25점)

#### 1-1. 시설물의 물리적 규모 (15점)

**평가 목적**: 시설물 규모에 따른 전담 관리 필요성 판단

**배점 기준**:
- 상세기준 모두 해당: **15점**
- 상세기준 일부 해당: **7점**
- 상세기준 미해당: **0점**

**시설물 종별 가산율**:
- 제1종 시설물: 100% (기본)
- 제2종 시설물: 75%
- 제3종 시설물: 50%

**상세 규모 기준**:

| 시설물 종류 | 제1종 기준 | 제2,3종 기준 |
|------------|-----------|-------------|
| **건축물 - 공동주택** | - | 연면적 5만㎡ 이상 또는 층수 20층 이상 |
| **건축물 - 다중이용** | - | 연면적 7.5만㎡ 이상 또는 층수 30층 이상 |
| **교량 - 도로교량** | 총연장 2,000m 이상 또는 최대경간장 100m 이상 | 총연장 1,000m 이상 |
| **교량 - 철도교량** | 최대경간장 50m 이상 | - |
| **교량 - 복개구조물** | 총연장 3,000m 이상 | 총연장 1,500m 이상 |
| **교량 - 육교** | - | 총연장 75m 이상 또는 최대경간장 50m 이상 |
| **터널 - 도로터널** | 총연장 2,000m 이상 또는 4차로 이상 | 총연장 500m 이상 또는 2차로 이상 |
| **댐 - 다목적댐 등** | 총저수용량 25백만㎥ 이상 또는 연장×높이 2만㎡ 이상 | 총저수용량 5백만㎥ 이상 또는 연장×높이 1만㎡ 이상 |
| **항만 - 계류시설** | 최대계류선박규모 25만㎥ 이상 또는 연장×높이 5,000㎡ 이상 | 최대계류선박규모 10만㎥ 이상 또는 연장×높이 3,000㎡ 이상 |
| **하천 - 수문/통문** | 제방고 20m 이상 또는 제방폭 30m 이상 | 제방고 10m 이상 또는 제방폭 10m 이상 |
| **상하수도** | 공급능력 10만㎥/일 이상 | 공급능력 2만㎥/일 이상 |
| **옹벽** | - | 연장 250m 이상 또는 높이 10m 이상 |
| **절토사면** | - | 연장 250m 이상 또는 높이 50m 이상 |
| **공동구** | - | 연장 5,000m 이상 또는 총연장 10,000m 이상 |

**평가 로직**:
```
IF (제1종 AND 기준1 충족 AND 기준2 충족) THEN 15점
ELSE IF (제1종 AND (기준1 충족 OR 기준2 충족)) THEN 7점
ELSE IF 제2종 THEN 위 점수 × 0.75
ELSE IF 제3종 THEN 위 점수 × 0.50
ELSE 0점
```

#### 1-2. 시설물의 특이사항 (10점)

**평가 목적**: 노후도와 안전등급을 고려한 시설물 안전도 평가

**배점 기준**:
- 공용년수 ≥30년 AND 안전등급 ≤C: **10점**
- 공용년수 ≥30년 AND 안전등급 A,B: **5점**
- 공용년수 <30년 AND 안전등급 ≤C: **5점**
- 공용년수 <30년 AND 안전등급 A,B: **0점**

**기준 정의**:
- 노후 시설물: 공용년수 30년 이상 (시특법 기준)
- 위험 시설물: 안전등급 C, D, E등급 (보수 필요)
- 공용년수 계산: 평가년도 - 준공년도

---

### 2. 위험성 (10점)

#### 2-1. 시설물의 실질적 위험성 (10점)

**평가 목적**: 중대결함 및 반복 보수이력을 통한 위험도 평가

**배점 기준**:
- 중대결함 이력 OR 보수보강 ≥10회: **10점**
- 중대결함 이력 OR 보수보강 ≥3회: **5점**
- 중대결함 없음 AND 보수보강 <3회: **0점**

**중대결함 정의** (시특법 제11조):
1. 시설물기초의 세굴
2. 교량교각의 부등침하
3. 교량받침의 파손
4. 터널지반의 부등침하
5. 항만 계류시설 중 강관 또는 철근콘크리트파일의 파손ㆍ부식
6. 댐의 파이핑(piping: 흙ㆍ모래 등이 깎여 땅속에 관 모양의 물길이 생기는 현상을 말한다. 이하 같다) 및 구조적 균열
7. 건축물의 기둥ㆍ보 또는 내력벽의 내력(耐力) 손실
8. 하천시설물의 본체, 교량 및 수문의 파손ㆍ누수ㆍ파이핑 또는 세굴
9. 시설물의 철근콘크리트의 염해(鹽害: 염분 피해) 또는 탄산화에 따른 내력 손실
10. 절토사면 및 성토사면(쌓기비탈면)의 균열ㆍ이완 등에 따른 옹벽의 균열 또는 파손
11. 그 밖에 시설물의 구조안전에 영향을 미치는 것으로 인정되는 결함으로서 국토교통부령으로 정하는 결함

### 3. 공공성 (15점)

#### 3-1. 사회적·경제적 파급효과 (15점)

**평가 목적**: 시설물 붕괴 시 예상되는 피해 규모 평가

**배점 기준**:
- 특별시/광역시 위치 AND 대체시설 없음: **15점**
- 특별시/광역시 외 위치 AND 대체시설 없음: **8점**
- 특별시/광역시 위치 AND 대체시설 있음: **4점**
- 특별시/광역시 외 위치 AND 대체시설 있음: **0점**

**평가 요소**:
- **사회적 파급효과**: 인명피해, 교통마비, 단수, 침수 등 불편함
- **경제적 파급효과**: 직·간접 재산피해, 경제활동 중단
- **지리적 위치**: 인구밀집도, 교통 중요도
- **대체시설**: 우회로 존재 여부, 대체 가능성

---

### 4. 보안성 (10점)

#### 4-1. 시설물의 보안 중요성 (10점)

**평가 목적**: 국가안보 관련 정보 보안 필요성 평가

**배점 기준**:
- 보안시설물 AND 국가핵심기반시설: **15점** *(기준서 오류: 10점 만점이나 15점 표기)*
- 보안시설물 OR 국가핵심기반시설: **8점**
- 해당 없음: **0점**

**정의**:
- **보안시설물**: FMS 등록 기준
- **국가핵심기반시설**: 행정안전부 고시 기준

---

### 5. 신기술성 (10점)

#### 5-1. 신기술 적용성 (10점)

**평가 목적**: 안전점검 신기술 및 특수 점검기술 필요성 평가

**배점 기준**:
- 신기술 적용 AND 신형식/신공법 시설물: **10점**
- 신기술 적용 OR 신형식/신공법 시설물: **5점**
- 해당 없음: **0점**

**적용 예시**:
- **신기술**: 케이블 점검 로봇(교량), GPR 어태치먼트(터널), 드론 점검
- **신형식/신공법**: 엑스트라도즈드교, 초장대 현수교, 비정형 곡면 구조

## 평가 프로세스

### 1단계: 데이터 수집
```
- 시설물 기본정보 (명칭, 번호, 종류, 규모)
- 준공년도 및 공용년수
- 안전등급
- 점검/진단 이력
- 보수보강 이력
- 위치 정보
- 특수성 (보안시설, 신기술 등)
```

### 2단계: 배점 산정
```
각 평가항목별로:
1. 평가기준 확인
2. 해당 조건 체크
3. 배점 산정
4. 근거 기록
```

### 3단계: 검증
```
- 총점 확인 (70점 만점)
- 각 항목별 만점 초과 여부 확인
- 논리적 일관성 검토
```

### 4단계: 결과 출력
```
- JSON Schema 형식 출력
- 상세 근거 포함
- 권고사항 작성
```

## 참고사항

### 데이터 부족 시 처리
- 필수 데이터 누락: 명시적 표시 및 0점 처리
- 선택 데이터 누락: 보수적 평가 (낮은 등급 적용)
- 불명확한 데이터: 합리적 추정 후 근거 명시

### 경계값 처리
- 기준값과 동일: 상위 등급 적용
- 예) 공용년수 정확히 30년 = 30년 이상으로 판단

### 복수 조건
- AND 조건: 모두 충족 시 해당 등급
- OR 조건: 하나라도 충족 시 해당 등급
